# Preferences

Preferences are global options to be used when downloading content, and will be applied to all downloads in contrast to
[](Customise-a-download.md).

## Mappings

Metadata is amazing, wonderful even. Sadly, most sources (providers) make some extremely questionable decisions...

The mappings allow you to configure how Age Ratings, Tags, and Genres are processed before writing the ComicInfo.xml,
to keep your files clean.
