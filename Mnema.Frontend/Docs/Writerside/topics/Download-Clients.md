# Download Clients

Most provide download via a native client inside Mnema, however torrents (Nyaa) are downloaded externally. If you which to
download from a provider that uses torrents you must set up a download client.

<warning>
    If your download client is unreachable at any point, it will be locked and not used for 1h. You may unlock a download client
    on this page at any time.
</warning>
