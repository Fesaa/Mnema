# Downloading

After choosing which providers to set up pages for in settings, you can use the pages to search for content to download. 
Each provider exposes its own filters.

![download_search_example.png](download_search_example.png)

## Results

Results are displayed in a grid with pagination if applicable. You can then decide to start your download as a one-off, 
or monitor the series. See [](Automatic-Downloads.md) for more information.

![download_results.png](download_results.png)

## Preview

Preview a series by pressing the info icon

<note>
    This data is not processed by Mnema, as it the raw data from the provider. 
</note>
