# Customize a download

When downloading you can decide a few more options before starting a download. These depend on the provider you're downloading
from. 

<tip>
    Mnema automatically check your files on disk to prevent duplicate downloads. If your naming is off, or Mnema does not correctly find the files
    turn off "Start Download Immediately" to manually select which files to download
</tip>

![download_option_general.png](download_option_general.png)

<warning>
    Scanlation groups can be used to only download if a specific group uploads. Their identification differs from providers
    some use the name, others the id.
</warning>
