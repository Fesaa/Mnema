
# Mnema

[![Tests](https://github.com/Fesaa/Mnema/actions/workflows/tests.yml/badge.svg)](https://github.com/Fesaa/Mnema/actions/workflows/tests.yml)

DOCS: https://fesaa.github.io/Mnema/

Mnema is a self-hosted Web Application for downloading content, primarily Manga.
Automatically including rich metadata, and a clean filesystem. While usable for any, the focus is on 
Kavita compatibility.
